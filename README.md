# A3mat v pomoyke

The best discord bot to annoy @a3mat

## Usage:
This command moves pinged users n times:
```
move <number> [users...]
```
This command moves pinged users 5 times:
```
move [users...]
```

## Hosting:
- Set enviroment variable `DISCORD_TOKEN` for your discord bot token. 
- Clone repo and run `cargo run`
